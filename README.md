# rn-masked-text-input

Masked text input for react native

## Installation

```sh
npm install rn-masked-text-input
```

## Usage


```js
import { RnMaskedTextInputView } from "rn-masked-text-input";

// ...

<RnMaskedTextInputView color="tomato" />
```
